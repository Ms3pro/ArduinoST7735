# ArduinoST7735
Дисплей сжирает все ресурсы процессора когда отрисовывает графику и поэтому CAN mcp2515 , серийные порты в момент отрисовки на дисплее бесполезная штука на этой плате мега , не хватает частоты процессора !!!
